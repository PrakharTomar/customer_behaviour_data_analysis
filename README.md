# 📊 Data Analytics Project

## 📌 Overview  
This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to delivering actionable business insights. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based querying, dashboard creation, and final reporting.

The objective is to transform raw data into meaningful insights using a combination of Python, SQL, and visualization tools.

---

## 📂 Dataset  
- The dataset contains structured data for business analysis  
- Includes both categorical and numerical features  
- Data cleaning was performed to handle missing values, duplicates, and inconsistencies  

---

## 🛠️ Tools & Technologies  
- **Python** – Data loading, cleaning, and EDA  
- **Pandas, NumPy, Matplotlib, Seaborn** – Data analysis & visualization  
- **SQL (PostgreSQL / MySQL / SQL Server)** – Querying and insights extraction  
- **Power BI** – Dashboard creation  
- **Gamma** – Presentation creation  
- **Jupyter Notebook** – Development environment  

---

## 🔄 Project Workflow  

### 1. Data Loading  
- Loaded dataset using Pandas  
- Checked structure, columns, and data types  

### 2. Exploratory Data Analysis (EDA)  
- Summary statistics  
- Distribution analysis  
- Trend and pattern identification  
- Data visualization  

### 3. Data Cleaning  
- Handled missing values  
- Removed duplicates  
- Standardized data formats  

### 4. SQL Analysis  
- Imported cleaned data into database  
- Performed:
  - Aggregations  
  - Filtering  
  - Grouping  
- Used PostgreSQL / MySQL / SQL Server  

### 5. Dashboard Creation  
- Built interactive dashboard in Power BI  
- Included KPIs, filters, and visual insights  

### 6. Reporting & Presentation  
- Created detailed report  
- Designed presentation using Gamma  

---

## 📊 Dashboard  
The Power BI dashboard includes:

- Interactive filters
